#Iconpacker Tools
一个给图标包作者用的小工具

目前实现的功能：
- 根据目录下的文件生成对应的Drawable.xml和Iconpack.xml(NanoIconPack)
- 联网查询Appfilter(感谢@By-syk)

目标功能：
- 全自动生成Drawable.xml\Iconpack.xml\Appfilter.xml
- 自动根据图标文件和Appfilter.xml生成MIUI/EMUI等主题的特殊命名的图标文件
- 自动压缩图标
